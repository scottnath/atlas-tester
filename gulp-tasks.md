# Useful Gulp Tasks

* ```gulp```
  * runs all *development* tasks and starts a dev server
* ```gulp help```
  * lists all available gulp tasks
* ```gulp build```
  * builds the distribution version of your app
* ```gulp deploy:ghPages```
  * deploys the build folder to gh-pages branch, pushes to github
